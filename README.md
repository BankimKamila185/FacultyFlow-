# FacultyFlow

A full-stack React and Node.js application for a faculty management portal.

## Project Structure

- `frontend/`: React application powered by Vite, utilizing an elegant custom CSS (glassmorphism/dark mode) dashboard.
- `backend/`: Node.js + Express API server.

## Getting Started

### Backend
1. `cd backend`
2. `npm install` (if not already installed)
3. `node server.js`
   - Runs on `http://localhost:5000`
   - Test endpoint: `http://localhost:5000/api/health`

### Frontend
1. `cd frontend`
2. `npm install` (if not already installed)
3. `npm run dev`
   - Start the Vite development server to view the rich React UI.

## Features
- Faculty Directory
- Department Statistics
- Sleek modern design with a responsive layout
